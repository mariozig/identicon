# Identicon

Generate [Identicons](https://en.wikipedia.org/wiki/Identicon) from Elixir.  Takes in a text string, hashes it and writes an identicon (in PNG) to the file system.

## Usage
```elixir
Identicon.main("David Hasselhoff")
```
